*Requires Android 6.0 or higher.*

## About KioDex

KioDex is a fork created to explore modern Android development and provide a refined user experience. It builds upon the solid foundation of Yokai and Tachiyomi.

## Features

<div align="left">

<details open="">
    <summary><h3>From KioDex</h3></summary>

* **Advanced Source Selector**: Proactively searches all sources for availability.
* **Modern UI**: Updated Home Page and Dialogs.
* **Jikan Integration**: Seamless manga discovery via MyAnimeList API.

</details>

<details>
    <summary><h3>From Yokai</h3></summary>

* NSFW/SFW library filter.
* New themes.
* ComicInfo.xml support.

</details>

<details>
    <summary><h3>From upstream (Tachiyomi)</h3></summary>

* Local reading of downloaded content.
* Configurable reader with multiple viewers.
* Tracker support (MyAnimeList, AniList, etc.).
* Categories to organize your library.
* Light and dark themes.
* Create backups locally or to cloud services.

</details>
